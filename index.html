TCP/IP 5 Layer Model

1. Application Layer

Application Layer

Protocols used: HTTP, FTP, SMTP

Responsibilities: Supports application services for end-user applications

2. Transport Layer

Transport Layer

Protocols used: TCP, UDP

Responsibilities: Provides reliable or unreliable data transport between devices

3. Network Layer

Network Layer

Protocols used: IP, ICMP, ARP

Responsibilities: Routes data through different networks using IP addresses

4. Data Link Layer

Data Link Layer

Protocols used: Ethernet, PPP, HDLC

Responsibilities: Provides reliable data transfer over the physical layer

5. Physical Layer

Physical Layer

Protocols used: Ethernet, Wi-Fi, Bluetooth

Responsibilities: Handles the physical connection between devices
